# Introduction to v-onboarding

v-onboarding is a powerful Vue 3 library for creating interactive product tours and onboarding experiences. Built with TypeScript, it provides a fully-typed API with smart tooltip positioning, SVG overlays, and extensive customization options.

## What is v-onboarding?

v-onboarding allows you to guide users through your application with step-by-step tours that highlight specific elements and display contextual information. Whether you're onboarding new users, showcasing features, or providing in-app tutorials, v-onboarding makes it simple and accessible.

## Key features

v-onboarding is designed to provide a complete onboarding solution with minimal setup:

## Use cases

v-onboarding is perfect for:

- **User onboarding**: Guide new users through your application's key features
- **Feature announcements**: Showcase new functionality to existing users
- **Interactive tutorials**: Create step-by-step walkthroughs for complex workflows
- **Contextual help**: Provide in-app guidance for specific tasks or features
- **Product tours**: Highlight your application's capabilities in a structured way

## Simple example

Here's a minimal example showing how easy it is to create an onboarding tour:

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { VOnboardingWrapper } from 'v-onboarding'
import 'v-onboarding/dist/style.css'

const wrapper = ref(null)
const steps = [
  {
    attachTo: { element: '#feature-1' },
    content: { title: 'Welcome!', description: 'Let me show you around.' }
  },
  {
    attachTo: { element: '#feature-2' },
    content: { title: 'Next Feature', description: 'Here is another feature.' }
  }
]
</script>

<template>
  <VOnboardingWrapper ref="wrapper" :steps="steps" />
  <button @click="wrapper?.start()">Start Tour</button>
  
  <div id="feature-1">Feature 1</div>
  <div id="feature-2">Feature 2</div>
</template>
```

<Note>
The library automatically handles tooltip positioning, overlay rendering, and keyboard navigation - you just define the steps!
</Note>

## Framework support

v-onboarding is built specifically for:

- **Vue 3**: Full support for Vue 3.2.21 and above with Composition API
- **Nuxt 3**: Dedicated module with auto-imports for components and composables
- **TypeScript**: First-class TypeScript support with comprehensive type definitions
