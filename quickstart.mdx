Get started with v-onboarding by creating a simple interactive tour in your Vue 3 application.

## Prerequisites

- v-onboarding installed (see [installation guide](/installation))
- A Vue 3 project set up
- Basic knowledge of Vue Composition API

## Create your first tour

<Steps>
  <Step title="Import the components">
    Import the v-onboarding components and styles in your component:

    ```vue
    <script setup lang="ts">
    import { ref } from 'vue'
    import { VOnboardingWrapper, useVOnboarding } from 'v-onboarding'
    import 'v-onboarding/dist/style.css'
    </script>
    ```
  </Step>

  <Step title="Define your steps">
    Create an array of steps that describe your tour. Each step targets an element and displays content:

    ```typescript
    const steps = [
      {
        attachTo: { element: '#welcome-btn' },
        content: {
          title: 'Welcome!',
          description: 'Click here to get started with our app.'
        }
      },
      {
        attachTo: { element: '#features' },
        content: {
          title: 'Features',
          description: 'Explore all the amazing features we offer.'
        }
      },
      {
        attachTo: { element: '#profile' },
        content: {
          title: 'Your Profile',
          description: 'Manage your account settings and preferences here.'
        }
      }
    ]
    ```
  </Step>

  <Step title="Add the wrapper component">
    Place the `VOnboardingWrapper` component in your template:

    ```vue
    <template>
      <div>
        <VOnboardingWrapper ref="wrapper" :steps="steps" />
        
        <!-- Your app content -->
        <button id="welcome-btn" @click="wrapper?.start()">
          Start Tour
        </button>
        
        <div id="features">
          <h2>Features</h2>
          <!-- Feature content -->
        </div>
        
        <div id="profile">
          <h2>Profile</h2>
          <!-- Profile content -->
        </div>
      </div>
    </template>
    ```
  </Step>

  <Step title="Start the tour">
    Control the tour programmatically using the wrapper ref:

    ```typescript
    const wrapper = ref(null)

    // Start the tour
    const startTour = () => {
      wrapper.value?.start()
    }
    ```
  </Step>
</Steps>

## Complete example

Here's a complete working example:

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { VOnboardingWrapper, useVOnboarding } from 'v-onboarding'
import 'v-onboarding/dist/style.css'

const wrapper = ref(null)

const steps = [
  {
    attachTo: { element: '#welcome-btn' },
    content: {
      title: 'Welcome!',
      description: 'Click here to get started with our app.'
    }
  },
  {
    attachTo: { element: '#features' },
    content: {
      title: 'Features',
      description: 'Explore all the amazing features we offer.'
    }
  },
  {
    attachTo: { element: '#profile' },
    content: {
      title: 'Your Profile',
      description: 'Manage your account settings and preferences here.'
    }
  }
]
</script>

<template>
  <div class="app">
    <VOnboardingWrapper ref="wrapper" :steps="steps" />
    
    <header>
      <button id="welcome-btn" @click="wrapper?.start()">
        Start Tour
      </button>
      <button id="profile">
        Profile
      </button>
    </header>
    
    <main>
      <section id="features">
        <h2>Features</h2>
        <p>Discover what makes our app great</p>
      </section>
    </main>
  </div>
</template>
```

## Using the composable

Alternatively, use the `useVOnboarding` composable for more control:

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { VOnboardingWrapper, useVOnboarding } from 'v-onboarding'

const wrapper = ref(null)
const { start, finish, goToStep } = useVOnboarding(wrapper)

const steps = [
  // ... your steps
]

// Control methods
const startTour = () => start()
const skipToEnd = () => goToStep(steps.length - 1)
const endTour = () => finish()
</script>
```

## Using template refs

Instead of CSS selectors, you can attach steps to Vue template refs:

```vue
<script setup lang="ts">
import { ref } from 'vue'
import { VOnboardingWrapper } from 'v-onboarding'

const wrapper = ref(null)
const featureButton = ref(null)

const steps = [
  {
    attachTo: { element: featureButton }, // Use the ref directly
    content: {
      title: 'Important Feature',
      description: 'This button is ref-based!'
    }
  }
]
</script>

<template>
  <VOnboardingWrapper ref="wrapper" :steps="steps" />
  <button ref="featureButton">Feature</button>
</template>
```

## Adding lifecycle hooks

Execute code before or after each step:

```typescript
const steps = [
  {
    attachTo: { element: '#settings' },
    content: {
      title: 'Settings',
      description: 'Customize your experience'
    },
    on: {
      beforeStep: async ({ index, step }) => {
        console.log(`About to show step ${index}`)
        // Load data, trigger animations, etc.
      },
      afterStep: async ({ index, direction }) => {
        console.log(`Finished step ${index}`)
        // Track analytics, cleanup, etc.
      }
    }
  }
]
```
